## Pixel List Page
---
User may navigate to Pixel List page shows all active pixels with their details. Users can view pixel name, ID/key, type, number of links, status, and can edit or delete each pixel.  

**Screenshot**  
![Pixels List Page](/images/pixel/pixel-index.png)

## Create New Pixel
---
Users can create a new pixel by clicking the `Create Pixel` button. A pop-up appears where users can select the pixel type, enter the pixel name, provide the pixel ID/key, enable or disable the pixel, and click `Create` to finalize it.

**Screenshot**  
![Create Pixel Pop-up](/images/pixel/pixel-create.png)

## Edit Pixel
---
Users can edit an existing pixel by clicking the `Edit` icon on the list page. The edit pop-up is identical to the create pop-up but displays pre-populated values that can be updated and saved.

![Edit Pixel Pop-up](/images/pixel/pixel-edit.png)
