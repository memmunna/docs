The Links feature allows users to create, manage, and track shortened URLs. Users can access the Links page from the navigation menu and view existing links, create new ones, edit, delete, and configure advanced options such as UTM parameters, password protection, expiration, targeting, and rotation.

### How can I access the Links page?
- From the navigation menu bar, click **Links**.  
- You will be redirected to the Shortened Links list page.  
- If there are no links, the page shows the option to create a new short link directly.  

![Links Access](/images/links/links-access.png)  
![No Links Available](/images/links/no-link.png)

### What does the Shortened Links page contain?
- **Search and Filters:** Filter links by date, group, pixel, or other criteria.
- **Columns and Actions:**
  - Short Link (URL)
  - Original Link Title & Favicon
  - Clicks / Unique Clicks
  - Group Name
  - Pixels
  - Status
  - Copy Icon (copies the link)
  - Visit Icon (opens link in a new tab)
  - Share Icon
  - 3-dot Menu (Edit, Visit, Statistics, Delete)  
![Links List](/images/links/link-index.png)

## Creating a Short Link
### How do I create a new short link?
- Click **Add Link** from the list page.  
- A modal opens to input the **Destination URL**.  
![Add Link Modal](/images/links/link-create.png)

- After entering a URL, additional optional fields appear:
  - Domain selection for short link
  - Alias update
  - UTM Builder
  - Enable **Password Protect this link**  
- Click **Create Link** to save.  

![Additional Fields](/images/links/link-create2.png)  
![Password Protected Option](/images/links/link-create3.png)

- **UTM Builder:** Click the UTM button to open a popup for:
  - Source, Medium, Campaign, Term, Content
  - Live link preview
  - Save to apply  
![UTM Builder Popup](/images/links/link-create4.png)

- After creation, a success modal shows:
  - Copy Link button
  - **More Settings** (redirects to Edit page)
  - Share options (social media icons)  
![Link Created Successfully](/images/links/create-complete.png)

### How do I share a link?
- Click the Share icon from the list or success modal.  
- All social sharing options appear.  
![Share Link Options](/images/links/share-link.png)

## What is the 3-dot menu?
Each link has a **3-dot menu** that provides quick access to actions:

- **Edit:** Open the link edit page/modal  
- **Visit:** Open the destination URL in a new tab  
- **Statistics:** View detailed link statistics  
- **Delete:** Permanently delete the link  

![3-dot Menu Screenshot](/images/links/3dot-menu.png)

## Managing Existing Links
### How can I edit a link?
- From the list, click the **Edit icon** or **More Settings**.  
- Update Destination Url, domain, alias, link title, select pixel, select groups
- Enable/Disable **Password Protect**
- Enable/Disable **Statistics Public Access**
- The edit modal shows pre-populated data. Update fields and click **Save Changes**.
![Edit Link Modal](/images/links/link-edit.png)

### How can I delete a link?
- Click the **Delete** option from the 3-dot menu.  
- A double confirmation popup appears. Click **Delete** to permanently remove the link.  
![Delete Confirmation](/images/links/delete-double-confirm.png)

---

### How do I visit a short link?
- Click the Visit icon or Visit option in the 3-dot menu.
- The link opens in a new tab.
- If password-protected, then password input option visible . After submit correct password then redirect to destination url; otherwise, direct redirect to destinetion url.
![Password Protected Visit](/images/links/password-protected.png)

---
